# StudentDashboard Management System

A JavaFX-based student management system with role-based dashboards for students, teachers, and administrators.  
It provides tools for managing marks, attendance, and user accounts with a simple and styled interface.

##  Features

 **Role-based dashboards**:
  - Students: View marks and attendance with charts
  - Teachers: Monitor attendance with summaries
  - Admins: Add/remove users, edit marks, track attendance, reset passwords, and search users

- Secure login system with role detection  
- Marks editor for individual subjects  
- Attendance tracker with date and status  
- Password reset popup with masked input  
- Search and filter users by name or role  
- File-based persistence using `Users.txt`, `marks.txt`, and `attendance.txt`  
- Styled UI using `style.css`  

##  Technologies Used

- JavaFX for GUI  
- Java I/O for file handling  
- CSS for styling  
- IntelliJ IDEA for development  

##  How to Run

1. Clone the repository  
2. Open the project in IntelliJ IDEA (or any Java IDE)  
3. Run `Main.java` to launch the application  

> Ensure `Users.txt`, `marks.txt`, and `attendance.txt` are present in the `src` folder for full functionality.
